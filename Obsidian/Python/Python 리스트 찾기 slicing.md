## INPUT
```python
a = 1, 2, 3, 4, 5, 6, 7

print()
print(a)

print(a[-1])
print(a[:-1])
print(a[-1:])
print(a[:-1:2])
```


## OUTPUT
```python
(1,2,3,4,5,6,7)

7
(1,2,3,4,5,6)
(1,2,3,4,5,6,7) ##-> 틀림
(1,3,5,7)


```

`a[-1:]` -> 왜 7이 나올까?
제일 끝에 있는 게 7이니까
그래서 `a[-1]` 의 경우에도 7이 나온다.

근데 그러면 `a[-1:-3]` 은 왜 불가능할까?

slicing의 경우 default 가 1이기 때문이다.

그래서 `a[-1:-3:-1]` 로 변경하면 올바른 값이 나온다.
