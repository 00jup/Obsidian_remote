자주 사용되는 주요 ping 옵션:

데이터 크기/개수
- -s: 패킷 크기 지정 
- -c: 보낼 ping 개수 지정

시간 관련
- -i: ping 간격(초)
- -t: 타임아웃 시간
- -W: 응답 대기 시간
- --apple-time: 현재 시간 표시 (Apple 전용)

네트워크 설정
- -m: TTL 값 지정
- -S: 출발지 주소 지정
- -b: 특정 인터페이스에 바인딩 (Apple 전용)
- -I: 인터페이스 지정

기타
- -v: 자세한 출력
- -Q: 조용한 출력
- -f: flood ping
- -D: 타임스탬프 표시
